# Watson Charts

## Dump Watson data in CSV

```
watson log --csv > data/watson.csv
```

## Clean data using Python script

```
python py-scripts/process_watson.py
```

## Run Node.js Http Server for the dashboard

```
npm run serve
```

## Check Dashboard

```
http://localhost:8000/
```
